Project Sample – API Ninjas Demo

This sample shows a working example of my Phase 4 assignment using the API Ninjas service. You can either: see a list of public holidays for a country, or fetch a random dad joke.

How this project works

This page integrates with the API Ninjas REST APIs using vanilla JavaScript and the modern fetch() API.

  • Public Holiday mode: calls /v1/publicholidays with a country code (for example AU or US) and lists all holidays returned.
  • Dad Joke mode: calls /v1/dadjokes and displays one random joke. The form enforces that only 1 joke is allowed (matching the Phase 4 requirements).
  • Error handling: HTTP errors, empty responses and invalid user input are all handled and displayed in the result area.

The behaviour on this page matches the Ninja API form that was originally implemented and tested in Phase 4 of the portfolio.
