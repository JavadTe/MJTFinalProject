Steam Game Update

Steam Game Update is a cross-platform mobile app built with .NET MAUI (C#) that displays live PC game data from the official Steam Store API. Users can browse Specials, Top Sellers, New Releases and Coming Soon through a modern, card-based interface with smart price formatting, readable discount dates and clear platform indicators.

HTML Website Project

A fully responsive multipage website built using HTML5, CSS3 and JavaScript. The project features a clean layout, semantic HTML for accessibility, consistent styling and a mobile-friendly design. It demonstrates core front-end skills including page structure, custom styling, responsive layouts and basic interactive behaviour.

IP, Ethics & Privacy Policy

A structured written project covering intellectual property, ethical use of technology and privacy responsibilities in IT. It explains key legal protections, professional behaviour expectations, user data safeguards and the importance of responsible digital practices. Demonstrates understanding of legal compliance, ethical decision-making and privacy principles in modern computing.
